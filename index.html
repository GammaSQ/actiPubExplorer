ActivityPub Validator

BEWARE:

Early alpha stages. Might very well cause bogus validation!

Known errors:

  • doesn't enforce url-structure for href-elements
  • doesn't enforce lang-format for language-maps

Validates only structure! Does not validate cross-dependencies!

Global valid/invalid generally trustworthy. Figuring something broke is easy.

Figuring out what broke is hard. Error pointers are less trustworthy.

This is implemented using jsonschema, big thanks and shoutout to it's developers!

Generally speaking, I assume the "type" sets the structure for the object, so don't fuck that one up!
